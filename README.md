# Nova Language

**Nova** is a systems programming language focused on explicit correctness, determinism, and control.

Nova’s design philosophy is simple:

> Correct code should be the default, not a burden.

The language emphasizes:
- Explicit error handling and control flow
- Permission-based mutability (no hidden aliasing or borrowing)
- Deterministic behavior across platforms
- Strong static guarantees without excessive ceremony

This organization hosts the core Nova language implementation and, over time, may also host:
- Tooling (compiler, formatter, language server)
- Standard and community libraries
- Package registry infrastructure
- Documentation and specifications

## Repositories

- **`nova-lang`** — The Nova language specification, compiler, and core tooling

## Status

Nova is under active development.  
Expect breaking changes while the language and tooling stabilize.
